# Wedding_Invitation_
How to Use
This website is designed for inviting guests to the wedding. To start using the website, simply open it in your web browser. You will see the details of the wedding event, along with options to RSVP.

How to Add Photos
Navigate to the photos directory in the repository.
Add your desired photos in JPEG or PNG format.
You can organize photos into subdirectories if needed (e.g., family, friends).
Ensure to name the photos clearly to reflect their content.
Deployment Information
Ensure that you have the latest version of the code from the repository.
Use a web server to host the website. You can use platforms like GitHub Pages, Vercel, or Netlify to deploy.
Follow the specific instructions on the platform you choose for deployment.
After deploying, test the website by accessing the provided URL.
Contact
For any questions or contributions, feel free to reach out to the repository owner.
